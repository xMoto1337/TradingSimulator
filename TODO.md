# Trading Simulator - Feature Ideas

## Chart & Analysis
- [ ] Indicators (SMA, EMA, RSI, MACD, Bollinger Bands)
- [ ] P&L overlay on chart (entry price horizontal line)
- [ ] Price alerts (notification when price hits a level)

## Trading Features
- [ ] Limit/stop orders (limit buy, stop loss, take profit)
- [ ] Position sizing calculator (risk % + stop loss = auto quantity)
- [ ] Trade journal/notes (attach notes to trades)
- [ ] Performance analytics (win rate, avg win/loss, profit factor, max drawdown, equity curve)

## UI/UX
- [ ] Keyboard shortcuts (quick buy/sell, switch timeframes with number keys)
- [ ] Dark/light theme toggle or accent color picker
- [ ] Sound effects (order fill, price alert)
- [ ] Drag-to-resize panels (adjustable sidebar widths)
- [ ] Add differnt charting styles (Neon Glowing Smooth line chart)

## Data
- [ ] News feed (relevant news for current symbol)
- [ ] Order book visualization (depth chart for crypto)
- [ ] Multiple portfolios (separate paper trading accounts)
